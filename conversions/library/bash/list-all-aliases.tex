\section*{List all shell aliases in a colored, formatted display}
\textbf{Author:} marcos \hfill \textbf{Date:} 2026-04-25

\noindent Lists all aliases defined in the user's bash shell configuration file in alphabetical order with colored, formatted output. Works with bash, zsh, and other shells on Linux and BSD systems.

\section*{Language}
bash

\section*{Category}
shell-utilities

\section*{Command}
\begin{lstlisting}
bash -c 'shell_conf="${1:-$HOME/.bashrc}"; [ -f "$shell_conf" ] || { echo "Configuration file not found: $shell_conf"; exit 1; }; grep -E "^[[:space:]]*alias[[:space:]]" "$shell_conf" | sed -E "s/^[[:space:]]*alias[[:space:]]*//" | sort -f | while IFS= read -r line; do alias_name=${line%%=*}; alias_cmd=${line#*=}; printf "\\033[1;34m%-20s\\033[0m -> \\033[1;32m%s\\033[0m\\n" "$alias_name" "$alias_cmd"; done' _
\end{lstlisting}

\section*{Explanation}
This command reads the user's shell configuration file from the path passed as the first argument (defaulting to \textasciitilde{}/.bashrc), extracts alias definitions, sorts them alphabetically by alias name, and prints the alias name in blue and the stored alias command in green. It preserves the command text as written after the equals sign, which keeps the direct shell syntax intact and avoids fragile quote-stripping logic.

\section*{Tags}
shell, alias, utilities, configuration, listing

\section*{Dependencies}
bash, grep, sed, sort


\section*{Arguments}
\begin{enumerate}
  \item \textbf{CONFIG\_FILE} (Optional): Path to shell configuration file (optional, defaults to \textasciitilde{}/.bashrc) \\ Default: \textasciitilde{}/.bashrc
\end{enumerate}

\section*{Notes}
\begin{itemize}
  \item The command works with bash, zsh, and other shells by specifying the appropriate configuration file
  \item Color codes: alias names are displayed in blue, commands in green
  \item Alias names are listed in alphabetical order
  \item The command text is shown exactly as stored after the equals sign
  \item Can be used directly or converted to a permanent alias
  \item Works on Linux and BSD systems
\end{itemize}

\section*{Warnings}
\begin{itemize}
  \item Only shows aliases defined in the specified configuration file
  \item Does not display aliases defined in sessions or other files
  \item May not work with non-standard alias definitions
\end{itemize}
