\section*{List all shell aliases in a colored, formatted display}
\textbf{Author:} marcos \hfill \textbf{Date:} 2026-04-25

\noindent Lists all aliases defined in the user's shell configuration file with colored, formatted output. Works with bash, zsh, and other shells on Linux and BSD systems.

\section*{Language}
bash

\section*{Category}
shell-utilities

\section*{Command}
\begin{lstlisting}
bash -c 'shell_conf=${1:-~/.bashrc}; if [ -f "$shell_conf" ]; then grep -E "^alias\s" "$shell_conf" | sed "s/^alias\s*//" | while IFS= read -r line; do alias_name=$(echo "$line" | cut -d"=" -f1); alias_cmd=$(echo "$line" | cut -d"=" -f2- | sed "s/^\(\s\)*\(\"\|\'\)\(.*\)\(\"\|\'\)\(\s\)*$/\3/"); printf "\033[1;34m%-20s\033[0m -> \033[1;32m%s\033[0m\n" "$alias_name" "$alias_cmd"; done; else echo "Configuration file not found: $shell_conf"; fi' _
\end{lstlisting}

\section*{Explanation}
This command detects the user's shell configuration file based on the SHELL environment variable, then parses the file to extract all alias definitions. It formats the output with colors to make alias names (blue) and their corresponding commands (green) easily distinguishable. The command accepts an optional argument for the configuration file path, defaulting to \textasciitilde{}/.bashrc if not specified.

\section*{Tags}
shell, alias, utilities, configuration, listing

\section*{Dependencies}
bash, coreutils, grep, sed


\section*{Arguments}
\begin{enumerate}
  \item \textbf{CONFIG\_FILE} (Optional): Path to shell configuration file (optional, defaults to \textasciitilde{}/.bashrc) \\ Default: \textasciitilde{}/.bashrc
\end{enumerate}

\section*{Notes}
\begin{itemize}
  \item The command works with bash, zsh, and other shells by specifying the appropriate configuration file
  \item Color codes: alias names are displayed in blue, commands in green
  \item Supports both single and double quoted alias definitions
  \item Can be used directly or converted to a permanent alias
  \item Works on Linux and BSD systems
\end{itemize}

\section*{Warnings}
\begin{itemize}
  \item Only shows aliases defined in the specified configuration file
  \item Does not display aliases defined in sessions or other files
  \item May not work with non-standard alias definitions
\end{itemize}
