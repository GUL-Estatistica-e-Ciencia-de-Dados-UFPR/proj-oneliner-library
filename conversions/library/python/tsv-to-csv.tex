\section*{Convert TSV (tab-separated values) file to CSV using Python stdlib}
\textbf{Author:} Marcos de Carvalho \hfill \textbf{Date:} 2026-04-25

\noindent Converts tab-separated values (TSV) to comma-separated values (CSV) using Python's standard library csv module.

\section*{Language}
python

\section*{Category}
data-processing

\section*{Command}
\begin{lstlisting}
python3 -c "import sys,csv; csv.writer(sys.stdout).writerows(csv.reader(sys.stdin, delimiter='\t'))"
\end{lstlisting}

\section*{Explanation}
The command reads TSV data from stdin, parses it with csv.reader using tab delimiter, and writes it to stdout using csv.writer with default comma delimiter. It preserves quoting and escaping rules.

\section*{Tags}
python, tsv, csv, data-conversion, text-processing

\section*{Dependencies}
python3


\section*{Arguments}
\begin{enumerate}
  \item \textbf{INPUT\_FILE} (Optional): TSV input file (use stdin redirection) \\ Default: stdin
  \item \textbf{OUTPUT\_FILE} (Optional): CSV output file (use stdout redirection) \\ Default: stdout
\end{enumerate}

\section*{Examples}
\begin{enumerate}
  \item \texttt{python3 -c "import sys,csv; csv.writer(sys.stdout).writerows(csv.reader(sys.stdin, delimiter='\textbackslash\{\}t'))" < data.tsv > data.csv} - Convert data.tsv to data.csv
  \item \texttt{cat data.tsv | python3 -c "import sys,csv; csv.writer(sys.stdout).writerows(csv.reader(sys.stdin, delimiter='\textbackslash\{\}t'))" | head -20} - Preview first 20 rows converted
\end{enumerate}

\section*{Output}
\begin{lstlisting}
col1,col2,col3
value1,value2,value3

\end{lstlisting}

\section*{Notes}
\begin{itemize}
  \item Handles quoted fields with tabs correctly
  \item Preserves newlines within quoted fields
  \item Works with any TSV dialect compatible with Python's csv module
\end{itemize}

\section*{Warnings}
\begin{itemize}
  \item Large files may consume memory as all rows are read into list
  \item Input must be valid TSV format
\end{itemize}
